# 图片托管平台

一个现代化的图片托管和分享平台，支持图片上传、AI 生成、社交分享等功能。

## 技术栈

- **框架**: Next.js 15
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI 组件**: shadcn/ui
- **状态管理**: React Hooks
- **图标**: Lucide Icons
